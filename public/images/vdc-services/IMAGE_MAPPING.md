# VDC Services Image Mapping Guide

Please save the provided images with these exact filenames in the `/public/images/vdc-services/` directory:

## Image Assignments:

1. **bim-design.jpg** - Architectural sketch drawing (first image)
   - Service: BIM Design Services
   - Description: Hand drawing architectural plans

2. **visualization.jpg** - Holographic building with engineer (second image)
   - Service: BIM Visualization Services
   - Description: Person pointing at holographic 3D building model

3. **clash-detection.jpg** - MEP coordination with X marks (third image)
   - Service: BIM Clash Detection
   - Description: Building section showing MEP clashes with checkmarks

4. **coordination.jpg** - Multi-layered building (fourth image)
   - Service: BIM Coordination Services
   - Description: Exploded view showing Architecture, Structure, MEP layers

5. **rebar-detailing.jpg** - Rebar beam detail (fifth image)
   - Service: Rebar Detailing Services
   - Description: Detailed rebar reinforcement in concrete beam

6. **revit-drafting.jpg** - Computer/tablet with BIM model (sixth image)
   - Service: Revit Drafting Services
   - Description: Digital device showing BIM floor plan

7. **4d-bim.jpg** - Building phases timeline (seventh image)
   - Service: 4D BIM Services
   - Description: Building with arrows showing different phases (Shaade, Shaows, Faase, Fracal, Conceptive, Simulation)

8. **5d-bim.jpg** or **quantity-takeoff.jpg** - Cost analysis (eighth image)
   - Service: 5D BIM Services / BIM Quantity Takeoff
   - Description: Building model with money symbols and cost chart

9. **sketch-to-render.jpg** - Sketch to photorealistic render (ninth image)
   - Service: BIM Visualization Services (alternative)
   - Description: Side-by-side comparison of sketch and rendered building

10. **design-iteration.jpg** - Multiple design pencils (tenth image)
    - Service: BIM Design Services (alternative)
    - Description: Building with multiple pencils around it showing design iterations

11. **bim-management.jpg** - Multi-screen coordination (eleventh image)
    - Service: BIM Management Services
    - Description: Person at desk with multiple BIM coordination screens

12. **facility-management.jpg** - (Use any suitable alternative or duplicate)
    - Service: BIM Facility Management Services
    - Suggestion: Can reuse coordination.jpg or bim-management.jpg
